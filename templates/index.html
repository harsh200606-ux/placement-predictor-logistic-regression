Student Placement Prediction

This website uses a real Logistic Regression model trained on student-profile data to predict whether a student is likely to be placed.

Model Details

  • Algorithm: Logistic Regression
  • Type: Binary Classification
  • Target: Placed / Not Placed
  • Backend: Flask + scikit-learn

How it works

1. Input Features

CGPA, aptitude, communication, internship, projects, backlogs, and certifications are taken as inputs.

2. ML Model

The Flask backend sends the values to a trained scikit-learn Logistic Regression model.

3. Prediction

The model returns a probability and final class: Placed or Not Placed.
